# Soho
A syntax theme for [Visual Studio Code](https://code.visualstudio.com) i

Optimized for SCSS, HTML, JS, TS, and PHP files.

Font used in screenshots is <a href="https://github.com/IBM/type">IBM Plex Mono</a>.
